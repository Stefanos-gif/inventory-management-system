# Basic Inventory Management System

## Description
A console application to manage a simple inventory system.

## Technologies Used
- C#
- Make sure you have .NET SDK installed. You can download it from Microsoft's official site.

## Setup Instructions
1. Clone the repository:
   ```bash
   dotnet new console -o InventoryManagementSystem
   cd InventoryManagementSystem
   git clone https://github.com/Stefanos-gif/inventory-management-system.git
   cd inventory-management-system

   #to run
   dotnet run
